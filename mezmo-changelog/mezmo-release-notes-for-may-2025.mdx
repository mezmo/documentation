Mezmo is pleased to announce new features and updates for the Mezmo Platform.

## Features

### Pipeline Sources and Destinations for Checkly

With the new [Source](/telemetry-pipelines/checkly) and[ Destination ](/telemetry-pipelines/checkly-destination)components for [Checkly](https://www.checklyhq.com/), you can both receive and send telemetry traces based on the OpenTelemetry standard for you Checkly instances. [Check out this blog post](https://www.checklyhq.com/blog/streamlining-telemetry-mezmo/) for more information about using Checkly with a Mezmo Telemetry Pipeline.
