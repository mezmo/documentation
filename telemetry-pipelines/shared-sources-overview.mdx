### What Are Shared Sources?
Shared Sources are global configurations in Mezmo that allow data to be transmitted across multiple pipelines simultaneously. Instead of duplicating data transmission for each pipeline, a Shared Source ensures the same data is shared, reducing redundancy and improving efficiency. This feature is ideal for reusing common data sources across teams or workflows, minimizing configuration effort and ensuring consistency.
